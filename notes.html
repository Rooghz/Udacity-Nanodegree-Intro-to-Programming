The Web

The Web (or World Wide Web) is a collection of documents and other files. Most of the documents are made using Hypertext Markup Language (HTML).

What is HTML?

HTML stands for Hypertext Markup Language. It's used to define the structure or layout of a web page.

Paragraph and List have optional closing tags.

When we leave off a closing tag, It will simply treat everything that comes after that point as part of the list.

Here is an example of a nested list:

  • Mammals
    1. Raccoons
    2. Gorillas
  • Reptiles
    1. Iguanas
    2. Cobras
  • Birds
    1. Ostriches
    2. Ravens

Indentation can help show the nested structure of the elements.

It can also make the code much easier to read!
